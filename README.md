# Codeforces Solutions

This repository contains C++ solutions to introductory and beginner-level Codeforces problems.

## Description

Each `.cpp` file corresponds to a single Codeforces problem and includes a standalone solution implementation.
The code is written in a simple style focused on learning core programming and problem-solving concepts.

## Contents

Current solved problems include:

- Watermelon
- Way Too Long Words
- Team
- Next Round
- Domino Piling
- Bit++
- Petya and Strings
- Boy or Girl
- Helpful Maths
- Word / Word Capitalization
- Stones on the Table
- Soldier and Bananas
- Bear and Big Brother
- Elephant
- Beautiful Matrix
- Nearly Lucky Number
- Wrong Subtraction

## How to Run

Compile and run any solution file with a C++ compiler, for example:

```bash
g++ -std=c++17 Watermelon.cpp -o Watermelon
./Watermelon
```
